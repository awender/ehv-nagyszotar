# ehv-nagyszotar
Az eszperantó-magyar és magyar-eszperantó nagyszótár Windows alkalmazásának portable kiadásai.

Az ingyenes, reklámmentes Windows-alkalmazás portable változatának hivatalos letöltési oldala.

A program mindkét irányban kereshető:

- eszperantóról magyarra;
- magyarról eszperantóra.

## Ajánlott telepítés

A Microsoft Store-os változat automatikusan frissül. A Store-hivatkozás a tanúsítás és publikálás után kerül ide.

## Portable változat

A portable kiadás nem igényel telepítést.

1. Töltsd le a ZIP-fájlt a Releases oldalról.
2. Csomagold ki egy tetszőleges mappába.
3. Indítsd el az `EhvDictionary.exe` fájlt.

A program teljesen offline használható.
